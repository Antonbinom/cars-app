# Cars fullstack app

## Используемы технологии:

Frontend:
- Vue3
- Composition Api
- Axios

Backend: 
- Express.js
- Sequalize
- PostgresQL

## Установка

```sh
npm install

yarn
```

### Запуск

```sh
npm run dev

yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build

yarn build
```
